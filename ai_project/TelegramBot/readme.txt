Telegram-бот (bot.py) это многофункциональный Telegram-бот, который может обрабатывать изображения
различными способами.

Особенности:
Пикселизация изображений
Конвертировать изображения в ASCII art
Применение фильтров к изображениям
Изменение размера изображений

Установка
Клонируйте этот репозиторий.
Создайте виртуальную среду и активируйте ее.
Установить зависимости:
pip install -r requirements.txt

Создайте файл .env в корневом каталоге и добавьте токен бота:
TELEGRAM_BOT_TOKEN=your-telegram-bot-token

Запустите бота:
Питон bot.py
/start

Использование:
Отправьте изображение боту, и он предоставит опции для пикселизации, преобразования в
ASCII-искусство, применения фильтров или изменения размера изображения.
